## About

Chat platform using NestJS, Socket.io

## Features
- Login
- Register
- Chat private
- Chat group
- Phone call
- Video call

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run dev

# production mode
$ yarn run start:prod
```
